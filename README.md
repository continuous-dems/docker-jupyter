# Continuous DEMs Jupyter Docker image

> [!NOTE]
> This docker image is for getting us started.
> Perhaps we should actually add continuous DEMs software to the official
> StratusGeo/CryoCloud image?

## Usage

Use the image:

```bash
ghcr.io/continuous-dems/cudem:${VERSION}
```

It's best to specify a specific hash or tag for `${VERSION}` to ensure you get what you
want.
If you just need to try it out real quick, you can use `latest`, but it's generally not
recommended.
